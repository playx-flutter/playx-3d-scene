The maximum camera translation speed in world units per second for FREE_FLIGHT mode. Defaults to 10.
read / write
flightMoveDamping ↔ double?
Applies a deceleration to camera movement in FREE_FLIGHT mode. Defaults to 0 (no damping). Lower values give slower damping times. A good default is 15.0. Too high a value may lead to instability.
read / write
flightSpeedSteps ↔ num?

Sets an additional matrix that shifts (translates) the projection matrix. The shift parameters are specified in NDC coordinates.
